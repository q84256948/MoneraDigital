# OpenSpec: 机构级客户控制台 (Dashboard)

## 1. 目标
为高净值客户提供一个专业、直观且安全的资产管理工作台，集成资产总览、借贷管理和安全中心。

## 2. 页面结构与功能
### 2.1 仪表盘布局 (Layout)
- **侧边栏 (Sidebar)**: 统一导航，包含概览、资产、借贷、投资、安全设置、对账单。
- **顶栏 (Header)**: 用户信息、通知中心、语言切换。

### 2.2 核心页面
1.  **概览 (Dashboard Index)**:
    - 资产总额统计（折线图展示趋势）。
    - 活跃借贷状态卡片。
    - 热门投资产品表现。
    - 市场行情简报。
2.  **资产总览 (Assets Overview)**:
    - 各币种比例（饼图）。
    - 资产列表（可用、冻结、自有/借入状态）。
3.  **安全设置 (Security Settings)**:
    - 密码修改。
    - 2FA 状态显示（模拟）。
    - 提币白名单管理。
4.  **消息中心 (Notifications)**:
    - 系统通知列表。

## 3. 技术实现
- **UI 组件**: 使用 Shadcn UI (Card, Table, Tabs, Chart, Sidebar).
- **图表**: 使用 `recharts` 进行数据可视化。
- **状态管理**: 初始阶段使用 Mock 数据，后续对接 API。
- **路由**: 使用 `react-router-dom` 的嵌套路由实现布局复用。

## 4. 验证标准
- 页面响应式支持（适配 iPad 与 PC）。
- 国际化支持（中英双语）。
- 权限控制：未登录用户访问 `/dashboard/*` 需重定向至登录页。
